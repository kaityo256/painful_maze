# うんざりするほど正解パスが長い迷路

## 概要

うんざりするほど正解パスが長い迷路を作る。

## 使い方

    $ make 

## 用法

    $ ruby maze.rb size(整数) threshold(実数)
    $ ruby maze.rb 5 0.5 

### Size

サイズ(整数)。できあがる迷路は2**(size+1)+1。
例えば 5に設定すると、65x65の問題が作成される。

### Threshold

正解パスの長さを決めるパラメータ。全体の面積に対する、正解パスが占める面積の割合。
大きいほど正解パスが長くなるが、その分だけ分岐が減って迷路として簡単になる。
最低で0.25(1/4)、最高は1.0だが、アルゴリズムの関係上、0.95程度までしかいかない。

## 出力

こんな感じの迷路ができる。
